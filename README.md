#Fluxible-Bootstrap-Gulp-Webpack

##Installation

1. `npm install`
2. `bower install`

##Running

1. Dev - `gulp`
2. Build - `gulp build`

Much thanks to [anuragsimgeker](https://github.com/anuragsimgeker) for his excellent [fluxible-bootstrap-boilerplate](https://github.com/anuragsimgeker/fluxible-bootstrap-boilerplate) repository.
